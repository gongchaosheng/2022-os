# Rustlinngs-options
## if let 用"="左边的作为值匹配右边的
```
fn main() {
        let a = Some(6);
        if let Some(6) = a{
            println!("hello");
        }
}
```
这是用`Some(6)`匹配`a`，成功就打印`hello`  
```
fn main() {
        let a = Some(6);
        if let Some(x) = a{
            println!("{}",x);
        }
}
```
如果`a`的值是`Some()`类型，就把`a`的值赋给`x`
